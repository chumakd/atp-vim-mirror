Intro

Automatic LaTeX Plugin (ATP) provides functions, commands, maps to edit LaTeX files in one of the greatest editors: Vim.
It is made for vim 7.3. You can see the long feature list.
The main purpose of this plugin is to make you free from compiling cicle. When ATP finds that the file has changed it fires compiler of your choice and makes your document in the background. All you have to do is write (and read).
ATP tries to stay out of your way, but this is a comprehensive plugin and some choices had to be made.
Maps are highly configurable (some of them you can be turn on/off temporarily).
ATP has a powerful Tab completion for commands, environments, closing environments, ... . It also completes labels by numbers (as seen in your output file).
Instead of folding ATP provides Table of Contents, which also can be used to copy (yank), move, paste sections.

ATP contains LaTeX Box plugin. It is developed here. It provides vim interface to latexmk, which is included in TeX Live TeX distribution. However you GNU/Linux distribution might package it seprately (debian, ubuntu, gentoo, fedora, ctan)

Download/Installation

You can download one of the stable versions or a snapshot.
Snapshots have recent bug fixes, are published more often, and they contain unstable new features. If you want to have the latest version you can use them. Also if you want to hep with testing. You can download vba file or tar.gz file. To install vba file all you have to do is open it with vim and run the following vim command ':source %'. The tar.gz file has to be unbundled in your ~/.vim directory. (Note: it is not contained in any directory!!!, test it with tar -tzf first) Don't forget to run ':helptags ~/.vim/doc', if you install from tar.gz bundle.
Here is a list of requirements.

You need to set :syntax on and :filetype plugin on in your vimrc file for ATP to work. If you want to have indentation you should also add :filetype indent on.

Updates

If you have ATP you can use :UpdateATP[!] to update ATP to the latest release (without bang "!") or to the latest snapshot (with bang "!") from inside Vim. If you have installed snapshot and you want to downgrade to latest stable release use :UpdateATP.
